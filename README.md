# ElleSteps - Ladies Shoes Shop Website

A beautiful, professional 2-page website for selling ladies' shoes with Contact Us and Order booking forms.

## 📁 Project Structure

```
new/
├── index.html          # Home page with featured products
├── contact.html        # Contact form page
├── order.html          # Order/Booking form page
├── style.css           # Beautiful responsive styling
├── script.js           # Form handling and interactivity
├── images/             # Product images folder (ADD YOUR IMAGES HERE)
└── README.md           # This file
```

## 🖼️ Adding Product Images

1. **Create an `images` folder** in your project directory (already created for you)
2. **Add your shoe product images** with these filenames:
   - `shoe1.jpg` - Hermes Sandals
   - `shoe2.jpg` - Casual Sneakers
   - `shoe3.jpg` - Elegant Boots
   - `shoe4.jpg` - Wedding Collection
   - `shoe5.jpg` - Speed Cat Heels
   - `shoe6.jpg` - Elegant Heels
   - `shoe7.jpg` - Summer Sandals
   - `shoe8.jpg` - Premium Loafers
   - `shoe9.jpg` - Ballet Flats
   - `shoe10.jpg` - Ankle Booties

3. **Image specifications:**
   - Recommended size: 300x300px or 400x400px
   - Format: JPG, PNG, or WebP
   - Keep file sizes under 500KB for faster loading

## 📋 Features

### Home Page (index.html)
- Beautiful hero section with call-to-action
- 10 featured product cards with images, names, and prices
- Responsive grid layout that works on all devices
- Smooth animations and hover effects

### Contact Page (contact.html)
- Email input field
- Message textarea
- Business contact information
- Store hours and location
- Form validation

### Order Page (order.html)
- Personal information section (First Name, Last Name, Email, Phone)
- Delivery address section (Street Address, Country)
- Order details section with:
  - Shoe type dropdown (pre-populated with all products)
  - Quantity selector
  - Ability to add multiple items
  - Remove item button

2. **Browse products**: See all featured shoe collections on the home page
3. **Contact**: Click "Contact Us" to send a message

## 🔧 Customization

### Change the Shop Name
Edit the logo text in all three HTML files:
```html
<div class="logo">👠 YourShopName</div>
```

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    ...
}
```

### Change Product Names/Prices
Edit the item cards in `index.html` and the dropdown options in `order.html`

### Change Contact Information
Edit the contact info in `contact.html`:
- Email
- Phone
- Store Address
- Hours
